# ♟️ World Chess Championship Analysis — Matplotlib & Seaborn

A data visualization project exploring game patterns, openings, 
and player performance across WCC history.

## 📌 About
This project explores a synthetically generated World Chess 
Championship dataset (500 games, 1886–2024) through 5 key visualizations.
Players included: Gukesh D, Magnus Carlsen, Viswanathan Anand,
Vladimir Kramnik, Garry Kasparov, Anatoly Karpov, Bobby Fischer, Boris Spassky.

## 📊 Visualizations Included
1. ⚪🔵 Win Rate by Color — Pie Chart
2. 🎯 Most Popular Openings — Bar Chart
3. ⚔️ Victory Type Breakdown — Horizontal Bar Chart
4. 📈 Game Length Distribution — Histogram with KDE
5. 🔥 Player vs Opening — Heatmap of Avg Moves

## 🛠️ Libraries Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ▶️ How to Run
Open the `.ipynb` file in Google Colab (Recommended) or Jupyter Notebook
